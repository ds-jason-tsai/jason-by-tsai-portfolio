## 🏗️ Project Architecture & Navigation

This portfolio is divided into several strategic sections, each optimized for search engine visibility and user conversion.

### 1. 🤖 Automated AI News Pipeline (Articles)
The backbone of the site's dynamic content.
- **Crawler**: Python-based scraper deployed on **Cloud Run**, fetching the latest AI/Data trends.
- **Analyzer**: Leveraging **Google Gemini 1.5 Pro** to summarize and translate articles into ZH/EN/JA.
- **Storage**: **Google BigQuery** acts as the data warehouse for historical article tracking and deduplication.
- **Automation**: Triggered daily at 06:30 TPE via **Cloud Scheduler**.

### 2. 💎 Premium Access (Reports)
Exclusive high-value assets including deep-dive reports and video courses.
- **ZH**: [付費專區](https://jason-by-tsai-portfolio.vercel.app/zh/reports) - 包含 Salesforce GTM 個案研究、NotebookLM 專家指南。
- **EN**: [Premium Access](https://jason-by-tsai-portfolio.vercel.app/en/reports) - Strategic case studies and multi-year social dataset assets.

### 3. 📊 Interactive Dashboards
Real-time BI solutions showcasing advanced data visualization capability.
- **GEO Monitoring**: Tracking Generative Engine Optimization trends.
- **Social Listening**: Insurance forum sentiment analysis via Looker Studio.
- **Procurement Analysis**: Custom Streamlit-based data modeling for business operations.

### 4. 🎨 Project Portfolio
Curated collection of professional projects across MarTech, FinTech, and Retail.
- **ZH**: [作品集](https://jason-by-tsai-portfolio.vercel.app/zh/portfolio) - MarTech 技術架構、Aflac 保險研究等實務案例。

## 🛠️ Technical Implementation

### **Frontend Stack**
- **Core**: [Next.js 16](https://nextjs.org/) (App Router), [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **SEO**: Dynamic JSON-LD, Multilingual routing (`/zh`, `/en`, `/ja`), and automated UTM tracking.

### **Backend & AI Pipeline**
- **AI Models**: [Google Gemini 1.5 Pro](https://ai.google.dev/)
- **Infrastructure**: [Google Cloud Run](https://cloud.google.com/run), [Google Cloud Scheduler](https://cloud.google.com/scheduler)
- **Database**: [Google BigQuery](https://cloud.google.com/bigquery)
- **Automation**: [n8n](https://n8n.io/) & custom Python scripts

### **Integration**
- **Payment Gateway**: [ECPay](https://www.ecpay.com.tw/)
- **Analytics**: [Vercel Analytics](https://vercel.com/analytics), Speed Insights

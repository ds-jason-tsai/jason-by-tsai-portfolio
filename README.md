## 🛠️ Technical Implementation

- **Core**: [Next.js 15](https://nextjs.org/) (App Router), [Tailwind CSS](https://tailwindcss.com/)
- **Data & AI**: [Google Gemini API](https://ai.google.dev/), [BigQuery](https://cloud.google.com/bigquery), [Google Looker Studio](https://lookerstudio.google.com/)
- **Integration**: [ECPay](https://www.ecpay.com.tw/) (Payment Gateway), [n8n](https://n8n.io/) (Automation)
- **Deployment**: [Vercel](https://vercel.com/) (Frontend), [Google Cloud Run](https://cloud.google.com/run) (Pipeline Crawler)

## 📈 SEO & Growth Strategy

This project implements:
1. **Multilingual Routing**: `/zh`, `/en`, `/ja` paths for global search ranking.
2. **Dynamic Metadata**: SEO Titles & Descriptions optimized for each language.
3. **Structured Data**: JSON-LD Person & WebSite schema to enhance Google Search Rich Snippets.
4. **Link Integrity**: Automated UTM tracking for all external and internal navigation.
